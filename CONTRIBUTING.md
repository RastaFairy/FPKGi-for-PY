# Guía de contribución

¡Gracias por tu interés en contribuir a FPKGi Manager! 🎉

## 🤝 Cómo contribuir

### Reportar bugs
1. Verifica que el bug no esté ya reportado en [Issues](../../issues)
2. Abre un nuevo Issue con:
   - Título descriptivo
   - Descripción detallada
   - Pasos para reproducir
   - Versión de Python y SO

### Sugerir mejoras
1. Abre un Issue con la etiqueta `enhancement`
2. Describe claramente la funcionalidad deseada
3. Explica por qué sería útil

### Hacer Pull Requests
1. Fork el repositorio
2. Crea una rama: `git checkout -b feature/mi-caracteristica`
3. Realiza tus cambios
4. Commit con mensajes descriptivos: `git commit -m "feat: Agregar nueva característica"`
5. Push: `git push origin feature/mi-caracteristica`
6. Abre un Pull Request describiendo:
   - Qué cambios hace
   - Por qué son necesarios
   - Issues relacionados (si existen)

## 📝 Estándares de código

- Usa PEP 8 para Python
- Comenta código complejo
- Incluye docstrings en funciones
- Mantén la compatibilidad con Python 3.8+
