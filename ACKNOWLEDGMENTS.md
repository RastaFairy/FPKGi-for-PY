# Acknowledgments

## Project Lineage

### FPKGi Manager - The Journey

This project represents a continuous evolution of game management tools across PlayStation platforms.

---

## 👨‍💻 Original Creators

### Bucanero
- **Role:** Original architect and creator
- **Platform:** PSP Homebrew
- **Contribution:** Established the core philosophy and user experience paradigm for unified game package management
- **Impact:** Created the foundation that would inspire versions across multiple platforms
- **Repository:** [FPKGi Manager (PSP Homebrew)](https://github.com/bucanero) (if available)

### ItsJokerZz
- **Role:** First modern console porter
- **Platforms:** PS4, PS5
- **Contribution:** Adapted the original concept to modern PlayStation architecture, maintaining the essence while leveraging newer console capabilities
- **Impact:** Extended FPKGi Manager's reach to contemporary gaming systems and user base
- **Repository:** [FPKGi Manager PS4/PS5](https://github.com/ItsJokerZz) (if available)

### RastaFairy
- **Role:** Python edition architect
- **Platforms:** Windows, Linux, macOS
- **Contribution:** 
  - Ported the complete feature set to Python for cross-platform desktop support
  - Integrated modern web scraping with Playwright and Chromium for OrbisPatches data retrieval
  - Implemented multilingual support (9 languages)
  - Added advanced features: parallel downloads, persistent icon caching, real-time availability verification
  - Created comprehensive documentation and user guides
- **Impact:** Made FPKGi Manager accessible to PC users and expanded its capabilities
- **Repository:** [FPKGi Manager Python Edition](https://github.com/RastaFairy/FPKGi-for-PY)
